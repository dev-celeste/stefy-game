# 106.9 Presents: Stefan's Big Day

A pixel-art choose-your-own-adventure game — a personal gift built as a single, offline, USB-friendly HTML file.

You wake up as **Stefan** on an ordinary morning. One choice — go to work or call out — splits the day into two adventures:

- **Branch 1 — Invasion at Foulk:** a normal maintenance shift at the old folks home turns strange when some residents turn out to be aliens. Stefan's metal scream and handyman skills save the day.
- **Branch 2 — The 106.9 Riddle Hunt:** a cozy day off becomes a citywide radio contest. Crack game/anime riddles and clear a keyboard-playable DDR minigame to win the grand prize.

Each branch has four endings (eight total), retro chiptune sound, and a 106.9 M&S DJ narrator.
